Para correr el programa para saber si un grafo es bipartita o no

se le pasará un archivo txt donde está representado el grafo

g++ tarea4_bipartito.cpp -o bip
./bip < grafo.txt
El grafo de entrada no es bipartito.

Cabe señalar que para correr con versiones recientes de C++ es necesario agregar la bandera -std=c++11 para compilar con versiones recientes de c++
quedando de la siguiente manera:

g++ -std=c++11 tarea4_bipartito.cpp -o bip

******************************************
Programa contenedores

usar la bandera -std=c++11 para compilar con versiones recientes de c++
g++ -std=c++11 tarea4_contenedores.cpp -o contenedor

No es necesario pasarle nada ya que los contenedores están en código duro en el códifo
